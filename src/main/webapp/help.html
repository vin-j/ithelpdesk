Help Desk Application Documentation

This document provides an overview of the Help Desk application modules, their purposes, and core functionalities.

Modules and Their Purpose

1. Incidents

Purpose: Log, view, and manage customer-reported issues or system incidents.

2. Login Accounts

Purpose: Manage user authentication and account details for secure access.

3. Customers

Purpose: Add, search, and manage customer information and profiles.

4. Reports

Purpose: Generate, view, and download detailed reports for incidents, customers, or system usage.

5. Files

Purpose: Upload, manage, and view files related to incidents or system activities.
